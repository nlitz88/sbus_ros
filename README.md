# sbus_interfaces
This package defines a ROS message to mock packets received from an SBUS receiver. SBUS
documentation can be found in the official [SBUS repo](https://github.com/bolderflight/sbus).

This package was created by following along with
https://docs.ros.org/en/jazzy/Tutorials/Beginner-Client-Libraries/Custom-ROS2-Interfaces.html
